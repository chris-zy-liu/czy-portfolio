    Prediction & Search

    Projects that use data to estimate, search, and test decisions under uncertainty.

    1. live

      Fig. 2 Apr–May 2026

      GT Bus Arrival Prediction

      I collected 1M+ GPS observations from Georgia Tech's public transit API by polling every 15 seconds for about a month. I labeled next-stop arrivals with geospatial buffering, engineered route-progress, headway, speed, traffic, and time features, and trained an XGBoost model on a chronological split. The model now powers a live dashboard for next-stop arrival estimates and historical route behavior.

      • Python
      • XGBoost
      • Pandas
      • Geospatial
      • Vultr

    2. Fig. 3 Aug 2025–May 2026

      Evolutionary Symbolic Regression

      A GPU-accelerated genetic-programming experiment for evolving interpretable trading rules over intraday cryptocurrency data. I built a stack-machine tree evaluator, CUDA / CuPy scoring path, Binance-to-Parquet data pipeline, multi-timeframe feature generation, and out-of-sample backtesting tools for tracking P&L, drawdown, and rolling Sharpe.
