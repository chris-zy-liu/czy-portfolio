    Selected work

    Applied ML, plus the data engineering and deployment around it.

    1. live

      Fig. 2 Apr–May 2026

      GT Bus Arrival Prediction

      I polled Georgia Tech's public transit API every 15 seconds for about a month to collect 1M+ GPS observations, labeled next-stop arrivals with geospatial buffering, and engineered route-progress, headway, speed, traffic, and time features. An XGBoost model trained on a chronological split predicts next-stop arrival, served through a live dashboard.

      • XGBoost

      • Geospatial
      • Vultr

    2. Fig. 3 Aug 2025–May 2026

      Evolutionary Symbolic Regression

      A GPU-accelerated genetic-programming engine (DEAP / CuPy) that evolves tree-based trading strategies for four market-regime specialists. I wrote a custom stack-machine tree evaluator and CUDA kernel to score thousands of programs over millions of 1-minute records, automated Binance ingestion to Parquet, engineered 17+ multi-timeframe indicators, and built out-of-sample backtests tracking P&L, drawdown, and rolling Sharpe.

      • DEAP
      • CuPy
      • CUDA
      • NumPy
      • Parquet

      GitHub ↗

    3. Fig. 4 May 2026–Present · the figure up top

      desmos.py

      An open-source Python package (published to PyPI) that compiles a strict subset of the Python AST into Desmos LaTeX / state JSON and emits self-contained interactive HTML graphs — geometry, sliders, buttons, tickers, recursion, and list comprehensions all translate. Unsupported constructs are rejected with line-specific errors. Validated with 37 golden-file tests across Python 3.10–3.14; the Mandelbrot explorer at the top of this page is ~40 lines.

      • Python
      • AST / compilers
      • PyPI
      • Setuptools

      GitHub ↗ PyPI ↗

    4. Fig. 5 Jan 2026–Present

      Classify — AI classroom platform

      A time-synchronized infinite canvas that analyzes each step of a student's work and returns targeted feedback. I built the Flask backend and Gemini integration, plus hierarchical aggregation that rolls step-level feedback up into assignment and class-performance summaries, persisted in PostgreSQL and deployed on AWS EC2 + Vercel. Started at a hackathon (4 → 6 contributors) and registered as a Georgia LLC.

      • Flask
      • Gemini API
      • PostgreSQL
      • AWS EC2
      • Vercel

    5. Fig. 6 AR / generative 3D

      DecoAR

      A Python backend that scrapes retailer furniture dimensions and generates 3D assets with Microsoft TRELLIS, then drops them into an ARKit / RealityKit prototype for placement in LiDAR-scanned rooms within seconds — preview furniture in your own space before buying.

      • Python
      • Microsoft TRELLIS
      • ARKit
      • RealityKit
      • Generative 3D
